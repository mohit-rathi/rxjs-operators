Merge Map Operator

Maps each value to an Observable, then flattens all of these inner Observables using mergeAll.
Example - 01
Map

of("Tech", "Blog", "News")
.pipe(map(res => of(`${res} Video Uploaded`)))
inner observable: delay(2000)

  • {{ item }}
Map + MergeAll

of("Tech", "Blog", "News")
.pipe(map(res => of(`${res} Video Uploaded`)), mergeAll())
inner observable: delay(2000)

  • {{ item }}
MergeMap

of("Tech", "Blog", "News")
.pipe(mergeMap(res => of(`${res} Video Uploaded`)))
inner observable: delay(2000)
